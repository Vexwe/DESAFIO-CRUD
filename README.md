# DESAFIO-CRUD (Completo)
Desafio: Em quanto tempo você consegue reproduzir uma aplicação simples fazendo CRUD, Create, Read, Update, Delete, usando suas tecnologias de preferência.
<img width="822" height="434" alt="image" src="https://github.com/user-attachments/assets/dc53c1d1-8214-41c6-bdf7-e8e7961778fd" />
Tecnologias usadas: React.js, Node.js, Mysql

Pré requisitos para funcionar: 
Xamp
Node.js
Mysql Workbench

Como instalar?
Após instalar todos os programas a cima,,abra o xamp, e de start no Mysql Server. Em seguida abra o frontend, e rode "npm install" e faça o mesmo com o backend, depois configure as portas e crie a database no MySQL workbench, o nome que está pré-configurado nos arquivos é "todolist".
